Final deliverable repository for a BCI-based RC car control project.

- final presentation
- final runtime Python code
- final Arduino sketch
- final PyTorch model
- final training notebook
- public EEG dataset subset for Participant2

## Signal classes
- `left`
- `right`
- `blink`
- `none`

## Control mapping
- `left` -> left turn
- `right` -> right turn
- `blink` -> forward
- `none` -> forward
